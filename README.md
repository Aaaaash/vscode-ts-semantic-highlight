# Semantic Highlight for TypeScript

TypeScript 语义化高亮 VS Code 插件
## Features
* TypeScript/TypeScript-React 文件(枚举, 方法参数, 局部变量)语义化高亮
* JavaScript/JavaScript-React 文件(方法参数, 局部变量)语义化高亮
